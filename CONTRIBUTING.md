# Contributing

Thanks for helping improve Aegis Moderation.

## Development

1. Fork and clone the repository.
2. Copy `.env.example` to `.env` and fill local values.
3. Run `make install`.
4. Run `make test` before opening a pull request.

## Quality

Run:

```bash
make lint
make test
```

Keep model weights, datasets, logs, and secrets out of commits.
